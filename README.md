# Digit Recognition (Logistic Regression) 
This project trains a Logistic Regression classifier to recognize handwritten digits (0–9) using scikit-learn’s built-in Digits dataset (8×8 grayscale images).

## What’s inside
- Loads Digits dataset (`load_digits`)
- Visualizes sample digits
- Train/test split
- Trains Logistic Regression (`max_iter=1000`)
- Evaluates accuracy
- Confusion matrix
- Shows example predictions (predicted vs actual)
